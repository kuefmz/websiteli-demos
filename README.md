# Websiteli Demos

This repository stores personalized demo websites generated for Websiteli prospects.

## Structure

- shared-template/ reusable static demo website template
- prospects/ one folder per approved prospect demo
- prompts/ reusable demo generation prompts
- docs/ operating notes and deployment checklist

## Safety rule

Demo sites are created for review and sales enablement. They should not copy protected brand assets or impersonate a prospect.
