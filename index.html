Websiteli sales demos

Personalized demo websites for potential clients.

This public GitHub Pages site hosts review-ready demo websites created for Websiteli outreach. Each prospect demo should live under /prospects/prospect-slug/ and include a clear note that it is a concept demo, not the prospect's official website.

Visit Websiteli

Daily workflow

The scheduled ChatGPT job selects one strong lead, creates a tailored static demo in this repo, pushes it to main, then GitHub Pages deploys it automatically.

Safety rule

Do not impersonate a business or copy protected brand assets. Use public facts only and label every demo as a Websiteli concept preview.

Email outreach

After deployment, the Gmail connector can send the prospect a short email with the demo link and the main Websiteli link.
